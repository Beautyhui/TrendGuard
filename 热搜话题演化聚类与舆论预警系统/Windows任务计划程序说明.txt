================================================================================
  用「任务计划程序」每隔 30 分钟自动抓取（微博 + B站 + 抖音）
  无需让 Python 连续运行数小时；每次只运行约几十秒后退出。
================================================================================

一、先手动试一次
----------------
1. 在资源管理器中**双击**（快速点两下）文件图标运行，不要只点一下文件名（那是重命名/选中）：
      crawl_all_platforms_once.bat
2. 会弹出黑色命令行窗口，依次出现 [1/3][2/3][3/3]；最后会显示「全部步骤已执行完毕」，
   并提示「按任意键关闭窗口」——**请按一下键盘任意键**，窗口才会关。
3. 若仍无窗口或 CSV 不变：打开项目里 data\crawl_log.txt ，看是否有 START/END 与各步 OK/FAIL。
4. 打开 data 文件夹，确认三个 CSV 里时间戳是否有新增行。

说明：
- 微博需在「用户环境变量」里配置 WEIBO_COOKIE（你之前已配置可略过）。
- 抖音若经常失败，可额外配置 DOUYIN_COOKIE（可选）。

二、创建计划任务（Windows 10 / 11）
------------------------------------
1. 按 Win 键，输入：任务计划程序 ，打开。

2. 右侧点「创建任务…」（不要用「创建基本任务」，便于设置重复间隔）。

【常规】
  名称：热搜三平台采集
  勾选：不管用户是否登录都要运行（若灰色，可只选「只在用户登录时运行」）
  配置：选择你的 Windows 版本（如 Windows 10）

【触发器】→「新建」
  开始任务：按预定计划
  设置：每天
  开始时间：例如 08:00:00（任意你希望第一次运行的时间）
  已启用：勾选

  下方勾选：重复任务间隔
    间隔：30 分钟
    持续时间：无限期（或选「1 天」并在第二天再自动延续——视系统界面而定；
              若只能选「1 天」，可改为「持续 24 小时」并勾选「如果过了计划开始时间，立即启动任务」）

  确定。

【操作】→「新建」
  操作：启动程序

  ——— 路径里有空格或括号时必看（例如「新建文件夹 (4)」）———
  若弹出对话框：「参数似乎已包括在程序文本框中…」并建议把路径拆成
    程序=E:\新建文件夹   参数=(4)\...
  请务必点【否(N)】，否则会一直跑失败。

  路径含空格/括号时，请用下面「方式 A」（最稳），不要用无引号的整段路径。

  方式 A（推荐，适合含空格的项目路径）：
  程序或脚本：
    C:\Windows\System32\cmd.exe
  添加参数（整行一条，路径改成你的；注意最外层一对引号、里面 bat 路径一对引号）：
    /c ""E:\新建文件夹 (4)\新建文件夹\项目二\crawl_all_platforms_scheduled.bat""
  「起始于」：
    E:\新建文件夹 (4)\新建文件夹\项目二

  方式 B（路径无空格时可用）：
  程序或脚本：
    E:\你的项目\crawl_all_platforms_scheduled.bat
  添加参数：（留空）
  「起始于」：项目目录

  说明：crawl_all_platforms_scheduled.bat 内部会带 silent，计划任务里不必再填 silent。

  ——— 若你坚持用 crawl_all_platforms_once.bat ———
  程序或脚本填：
    C:\Windows\System32\cmd.exe
  添加参数（路径改成你的）：
    /c ""E:\新建文件夹 (4)\新建文件夹\项目二\crawl_all_platforms_once.bat" silent"
  「起始于」仍填项目目录。

  确定。

【条件】（可选）
  若希望仅插电时运行：勾选「只有在计算机使用交流电源时才启动此任务」。
  若希望休眠时也能唤醒：勾选「唤醒计算机运行此任务」（笔记本是否生效因机型而异）。

【设置】
  建议勾选：如果过了计划开始时间，立即启动任务
  若任务失败：可按需设置「如果任务失败，重新启动间隔」

3. 确定保存后，在任务列表中右键该任务 →「运行」，检查 data 下 CSV 是否新增行。

三、常见问题
-----------
Q：提示找不到 python？
A：安装 Python 时勾选 Add to PATH；或把 bat 里三处 python 改成 py（Windows 启动器）。

Q：笔记本合盖后不执行？
A：在电源选项里允许「睡眠时唤醒定时器」，并在计划任务「条件」里尝试勾选唤醒；或改为「仅登录时运行」且合盖改为不休眠。

Q：任务计划程序里右键「运行」界面好像没反应？
A：界面常仍显示「准备就绪」；请看任务下方「上次运行时间」是否更新、
   「上次运行结果」是否为 0x0；并打开 data\scheduled_invoke.txt（由
   crawl_all_platforms_scheduled.bat 写入）是否出现 started/finished 行；
   再看 data\crawl_log.txt 是否有新的 START/END。

Q：在「添加参数」里填 silent 会报错？
A：不要用主 bat + 参数。请改用「推荐」里的 crawl_all_platforms_scheduled.bat（参数留空）；
   或改用 cmd.exe + /c 那一行写法。

Q：能否补抓「过去 6 小时」的历史？
A：不能。热榜接口通常只给当前一版；历史只能靠定时任务从「现在起」不断 append 积累。
